# Medico

Medico is a full-stack healthcare application with backend APIs and frontend interface for managing users, appointments, and services.

## Features
- User authentication (login/register)
- Appointment booking system
- Admin dashboard
- Backend API (Node.js + MySQL)

## Tech Stack
- Frontend: Flutter / Web
- Backend: Node.js (Express)
- Database: MySQL

## Project Structure
- /backend → API, server, database
- /frontend → UI application

## Setup
1. Install dependencies
2. Configure `.env`
3. Run backend server
4. Run frontend app
